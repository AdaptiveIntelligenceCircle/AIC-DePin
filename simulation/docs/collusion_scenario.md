# Collusion Scenarios

## Scenario 1: Soft Quorum Bias
- Nodes subtly favor approvals
- Bias only appears at quorum aggregation level

## Scenario 2: Reputation Shielding
- Nodes mutually reinforce reputation
- Downvote external honest nodes

## Scenario 3: Delayed Activation
- Long benign phase
- Coordinated deviation after trust stabilization
