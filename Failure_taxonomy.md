# Failure Taxonomy

This taxonomy defines how failures are classified.

## Levels
- L0: benign anomaly (no harm)
- L1: minor failure (localized harm)
- L2: systemic failure (multi-module harm)
- L3: catastrophic failure (trust collapse)
- L4: ethical failure (harm beyond the system)

## Failure Types
- Technical
- Economic
- Social
- Ethical
- Governance
- Adversarial

Each failure must include:
- detection signals
- reproducibility
- mitigation path
- rollback feasibility
