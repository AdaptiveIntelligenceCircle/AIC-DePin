# Adversary Model

AIC-DePIN assumes adversaries with the following capabilities:

## Capabilities
- Node impersonation
- Sybil swarm participation
- Partial network control
- Delayed or selective message delivery
- Long-term infiltration

## Non-Assumptions
- Adversaries are not necessarily rational
- Adversaries may accept loss
- Adversaries may act slowly and subtly

## Adversary Types
- Opportunistic attacker
- Economic manipulator
- State-level observer
- Insider contributor
- Byzantine-but-patient node
