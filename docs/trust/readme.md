# Trust System in AIC-DePIN

This directory documents how trust is defined, evaluated, degraded,
and recovered within AIC-DePIN.

Trust in AIC-DePIN is:
- Behavioral, not declarative
- Contextual, not global
- Time-dependent, not static
- Reversible, not absolute

AIC-DePIN does not attempt to identify "good nodes".
It attempts to **limit the influence of untrusted behavior**.
