# How to Simulate in AIC-DePIN

Simulation in AIC-DePIN is not ad-hoc experimentation.
It follows a structured process.

## Step 1: Define Scenario
- Target subsystem (trust, routing, policy, sync)
- Expected failure mode
- Adversary profile

## Step 2: Configure Sandbox
- Resource limits
- Network conditions
- Initial trust states

## Step 3: Inject Behavior
- Node actions
- Message patterns
- Timing anomalies

## Step 4: Observe
- Trust evolution
- Policy decisions
- Routing changes
- Resource usage

## Step 5: Replay
- Reconstruct execution
- Compare alternative policies
- Identify divergence points

## Step 6: Iterate
- Adjust parameters
- Combine adversaries
- Escalate complexity

If a simulation produces no insight,
it is considered incomplete.
