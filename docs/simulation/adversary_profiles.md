# Adversary Profiles

Simulation assumes intelligent and patient adversaries.

## Profile Categories

### 1. Naive Attacker
- High noise
- Obvious malicious behavior
- Short-term impact

### 2. Economic Manipulator
- Incentive gaming
- Trust inflation
- Resource exhaustion

### 3. Slow Infiltrator
- Long-term benign behavior
- Gradual deviation
- Trust poisoning

### 4. Coordinated Swarm
- Multiple identities
- Shared timing patterns
- Distributed influence

### 5. Insider Adversary
- Access to code or configuration
- Subtle policy abuse
- Delayed exploitation

Profiles are combined and evolved over time.
