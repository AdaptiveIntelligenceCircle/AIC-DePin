# Postmortem Template

## Summary
What happened?

## Timeline
List key events with event_id and timestamps.

## Root Cause
Technical + governance + incentive factors.

## Detection
How was it detected? What failed to detect it earlier?

## Response
What actions were taken?

## Recovery
What was restored and what was lost?

## Lessons
What assumptions were wrong?

## Preventive Actions
Policy updates, code patches, governance changes.

## Ethical Reflection
Did the system cause harm beyond itself?
